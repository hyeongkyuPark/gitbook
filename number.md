# 숫자\(Number\)

## 📕 Number\(숫자\)

* JavaScript에서는 정수와 소수를 통틀어서 Number라는 데이터 타입으로 정의하고 있음 \(typeof\(\)를 통해 데이터 타입을 알아볼 수 있음\) 

```markup
<h1>Number 숫자타입</h1>
<script>
    document.write(1); // 1
    document.write('<br />'); // 개행 문자
    document.write(3.15); // 3.15
    document.write('<br />');
    document.write(typeof(1)); // number
    document.write('<br />');
    document.write(typeof(3.15)); // number
</script>
```
